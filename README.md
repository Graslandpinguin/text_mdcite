# text_mdcite
MoinMoin - Parser for a Markdown - Citation

    Creates my own citation style with markdown. Reduces typing.
##Dependencies
 * markdown library

##Syntax:

To use in a code block:
    
    {{{#!text_mdcite
    <Author>
    <Link>
    <Markdown citationtext here>
    }}}
